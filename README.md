# Bangla2Flow

**`Bangla2Flow`** হলো একটি IDE যা বাংলা কোডকে **`Flowchart`** এবং **`JavaScript`** আউটপুটে রূপান্তর করে।  
এটি শিক্ষার্থীদের জন্য এবং যারা বাংলায় কোডিং করতে চায় তাদের জন্য খুবই সহজ।

---

##  বৈশিষ্ট্যসমূহ 

- বাংলা কোড লেখা যায়  
- JavaScript আউটপুট দেখা যায়  
- Flowchart জেনারেট করা যায়  
- সাধারণ লুপ, শর্ত, ফাংশন, অ্যারে, স্ট্রিং অপারেশন সমর্থিত
