
# Bangla2Flow

Bangla2Flow হলো একটি IDE যা **বাংলা কোডকে Flowchart** এবং **JavaScript আউটপুট**-এ রূপান্তর করে।  
এটি শিক্ষার্থীদের জন্য এবং যারা বাংলায় কোডিং করতে চায় তাদের জন্য খুবই সহজ।
